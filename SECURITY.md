# Security Policy

## Reporting a Vulnerability

If you discover a security issue in this repository, report it privately by email:

- <mailto:shahulofficial16@gmail.com>

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

## Scope

This is a static portfolio website. Typical security concerns include:

- Unsafe third-party scripts
- Open redirects
- Injection risks in future dynamic integrations
- Exposure of sensitive data in repository history

## Response Process

- Initial response target: within 5 business days
- Valid reports are investigated and patched as quickly as possible
- Public disclosure can follow after remediation is available
