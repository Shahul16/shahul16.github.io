# Shahul Hameed Portfolio Website

Production-ready personal portfolio website for Shahul Hameed, focused on AI engineering, automation, cloud infrastructure, and IT systems delivery.

## Live Website

- Primary: <https://shahulhameed.site>
- GitHub Pages: <https://shahul16.github.io>

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Static hosting on GitHub Pages

## Project Structure

- `index.html` - main website markup
- `css/styles.css` - core styles
- `css/vendor.css` - third-party CSS
- `js/main.js` - site behavior and interactions
- `js/plugins.js` - third-party libraries bundle
- `images/` - portfolio/media assets
- `robots.txt`, `sitemap.xml`, `site.webmanifest` - SEO/PWA metadata
- `CNAME` - custom domain mapping for GitHub Pages

## Local Development

Since this is a static site, open `index.html` directly in a browser or run a local static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deployment

This repository is configured for GitHub Pages user-site deployment (`main` branch).

1. Push changes to `main`
2. Verify GitHub Pages build in repository settings
3. Confirm domain and HTTPS in Pages settings
