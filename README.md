## About

Personal portfolio built with pure HTML5, CSS3, and Vanilla JS — no frameworks, no build step.
Self-contained single-file design with dark editorial aesthetic.

## Stack

- HTML5 / CSS3 / Vanilla JS
- GitHub Pages + custom domain (CNAME)
- PWA-ready (manifest, favicons, service worker)
- Schema.org JSON-LD structured data
- SEO: sitemap.xml, robots.txt, canonical, OG tags
