# Next Portfolio (Modern Version)

Modern recruiter-focused portfolio implementation for Shahul Hameed.

## Positioning

- Technology Business Analyst
- Systems, Data, and Digital Transformation
- IT infrastructure and operations background

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality Checks

```bash
npm run lint
npm run build
```

## Key Files

- `src/app/page.tsx` - full portfolio page content and structure
- `src/app/globals.css` - visual system, tokens, and component surface styles
- `src/app/layout.tsx` - metadata, fonts, and global layout shell
- `public/Shahul_Hameed_ATS_Resume.txt` - ATS-friendly downloadable resume snapshot

## Notes

- Messaging is intentionally aligned to analyst/transformation roles.
- Avoid role inflation (no unsupported senior/principal architect labels).
